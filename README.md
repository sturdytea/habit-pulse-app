# 🍃 HabitPulse

Мобильное приложение на **SwiftUI** для отслеживания ежедневных привычек: цели на неделю, отметки по дням, серия и простая аналитика. Данные хранятся **локально** на устройстве.

## Возможности

- **Главная** — дата, прогресс за сегодня и быстрый переход к привычкам.
- **Список привычек** — создание, удаление (свайп), переход к деталям.
- **Деталь привычки** — отметка за сегодня, полоска недели, заметка, серия и число дней с отметками; редактирование и удаление из меню.
- **Инсайты** — сводка, недельный прогресс по каждой привычке, «лидер» по числу отметок.
- **Настройки** — о приложении, сброс данных, экран справки.
- **Загрузочный экран** — анимация (появление логотипа, пульс фона, вращающаяся дуга, полоса прогресса, индикатор загрузки), затем переход к основному интерфейсу.

## Требования для сборки

- macOS с установленным **Xcode** (рекомендуется актуальная стабильная версия).
- Для запуска на физическом устройстве: Apple ID и выбранная **Team** в настройках подписи таргета.

## Как запустить

1. Откройте файл проекта:

   `HabitPulse.xcodeproj`

2. В навигаторе схем выберите **HabitPulse** и симулятор или устройство.
3. Соберите и запустите проект: **⌘R** (или **Product → Run**).

Сборка из терминала (пример):

```bash
cd HabitPulse
xcodebuild -scheme HabitPulse -destination 'platform=iOS Simulator,name=iPhone 16' build
```

Подставьте имя симулятора из списка: `xcodebuild -scheme HabitPulse -showdestinations`.

## Данные и первый запуск

- Ключ в `UserDefaults`: `habitpulse.habits.v1`.
- При **первом** запуске (когда ещё нет сохранённого файла данных) приложение добавляет несколько **демо-привычек**, чтобы можно было сразу оценить интерфейс.
- **Настройки → Сбросить все привычки** очищает список и сохраняет пустое состояние; при следующем запуске демо-данные **не** подставляются снова.

## Иконка и внешний вид

- Иконка приложения лежит в `Assets.xcassets/AppIcon.appiconset` (формат для каталога: **1024×1024**).
- Общая стилистика: бирюзово-изумрудная палитра, карточки со скруглением, системные **SF Symbols** где уместно.
- Вёрстка рассчитана на разные размеры экранов (в т.ч. узкие — через `ScrollView`, ограничения по строкам и масштабирование текста).
